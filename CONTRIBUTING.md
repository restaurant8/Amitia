# Contributing / 贡献指南

感谢你关注 Amitia。为保护项目和贡献者，请遵守以下规则。

## 贡献要求

- 所有贡献必须来源合法。
- 贡献者必须有权提交代码、文档和素材。
- 不得提交未经授权复制的代码。
- 不得提交来源不明的 AI 生成代码。
- 不得提交违反第三方许可证的内容。
- Amitia 社区版本使用 AGPL-3.0-only。
- 实质性代码贡献在合并前需要签署 CLA。
- 未完成 CLA 的 PR 可以讨论，但不应合并。
- 贡献者必须保留第三方版权声明和许可证声明。
- 安全漏洞不要通过公开 Issue 披露，应遵循 SECURITY.md。
- 贡献者需要通过现有测试、格式化、Lint 和构建检查。
- 贡献行为不授予 Amitia 商标、Logo、角色形象或其他品牌资产使用权。

## Pull Request 检查项

- [ ] 我确认有权提交本次贡献。
- [ ] 本次贡献不包含未经授权的第三方代码或素材。
- [ ] 我已保留所有必要的第三方版权和许可证声明。
- [ ] 我已阅读并同意项目贡献规则。
- [ ] 如本次贡献属于实质性代码贡献，我已签署 CLA。

## English Summary

All contributions must be lawful and submitted by contributors who have the right to provide them. Do not submit unauthorized third-party code, unclear AI-generated code, or content that violates third-party licenses. Amitia Community Edition is licensed under AGPL-3.0-only. Substantial code contributions require a CLA before merge. Security vulnerabilities should be reported according to SECURITY.md rather than through public issues.
